# Project System Rules

## UI/UX Design & Implementation
Whenever you are asked to create, modify, or interact with ANY Frontend code (including Flutter UI components, screens, layout, or styling), you MUST:
1. Prioritize reading the `ui-ux-pro-max` skill instructions (`.agents/skills/ui-ux-pro-max/SKILL.md`).
2. Utilize the skill's design system generation capabilities to suggest matching styles, color palettes, typography, and UX guidelines before starting implementation.
3. Ensure the code adheres to the UI/UX best practices and stack guidelines recommended by the UI UX Pro Max toolkit.
