## 🎯 Tổng quan mục tiêu
Bản phát hành này tập trung vào việc kiện toàn kiến trúc dữ liệu thông qua việc chuẩn hóa các thực thể (entities) và tối ưu hóa cấu hình hệ thống. Những thay đổi này giúp cải thiện tính nhất quán của dữ liệu, đơn giản hóa quy trình quản lý cấu hình môi trường và nâng cao độ ổn định cho các bài kiểm thử tự động.

## ✨ Chi tiết thay đổi

### 🚀 Tính năng mới
- <kbd>fc731e1</kbd> **Nâng cấp thực thể Media**: Tích hợp Lombok để giảm mã nguồn dư thừa, chuyển đổi định danh sang Long và bổ sung lớp MediaMetadata để quản lý thông tin bổ trợ.
- <kbd>fc2aac4</kbd> **Khởi tạo dữ liệu Album**: Thiết lập các thực thể Album, AlbumMedia và repository tương ứng để hỗ trợ chức năng quản lý album.
- <kbd>95c2947</kbd> **Định danh AlbumMedia**: Bổ sung import UUID cho thực thể AlbumMedia nhằm đảm bảo tính định danh duy nhất.
- <kbd>21e56e7</kbd> **Cấu hình môi trường**: Thiết lập file `.env`, bổ sung DotenvConfig và cập nhật datasource để hỗ trợ các bài kiểm thử tự động với H2 database.

### 🛠 Bảo trì hệ thống
- <kbd>7bd0489</kbd> **Chuẩn hóa ID Album**: Chuyển đổi kiểu dữ liệu định danh (id) của thực thể Album từ UUID sang Long để thống nhất kiến trúc.
- <kbd>75967a0</kbd> **Tự động hóa timestamp**: Cập nhật các lớp thực thể sử dụng `@CreationTimestamp` để tự động ghi lại thời gian khởi tạo.
- <kbd>17a45b3</kbd> **Tối ưu hóa môi trường**: Cố định phiên bản Java 17 và tinh giản cấu trúc thực thể AlbumMedia.
- <kbd>e3ff207</kbd> **Hoàn thiện thực thể Media**: Tối ưu hóa cấu trúc thực thể Media sau khi áp dụng Lombok và các thay đổi về JdbcTypeCode.
- <kbd>f3b5869</kbd> **Hoàn thiện cấu hình**: Đóng gói các cấu hình .env, làm sạch mã nguồn AlbumMedia và bổ sung các dependency cần thiết cho H2 test.
- <kbd>b5dbab9</kbd> **Đồng bộ hóa**: Hợp nhất các thay đổi mới nhất từ nhánh remote master.
- <kbd>eefa3fc</kbd> **Đồng bộ hóa**: Hợp nhất các thay đổi mới nhất từ nhánh remote master.
