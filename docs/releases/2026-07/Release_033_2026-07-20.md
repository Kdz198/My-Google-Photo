## 🎯 Tổng quan mục tiêu
Bản phát hành này tập trung vào việc hoàn thiện tính năng hiển thị bộ sưu tập ảnh trên nền tảng Web thông qua việc tích hợp API và tối ưu hóa quy trình tải tệp tin lên hệ thống. Ngoài ra, chúng tôi đã cải thiện khả năng tương thích giữa frontend và backend, đồng thời tự động hóa quy trình tạo client API để nâng cao hiệu suất và tính ổn định của hệ thống.

## ✨ Chi tiết thay đổi

### 🚀 Tính năng mới
- <kbd>1ce3e90</kbd> **Tích hợp API thư viện ảnh**: Hoàn thiện màn hình hiển thị chính của bộ sưu tập ảnh trên giao diện Web với khả năng kết nối dữ liệu từ API.
- <kbd>6ad2430</kbd> **Tự động hóa CI**: Thiết lập quy trình CI và cấu hình tự động tạo mã nguồn client từ OpenAPI để đồng bộ hóa dữ liệu nhanh chóng.

### 🐛 Sửa lỗi
- <kbd>7f97b76</kbd> **Cấu hình MIME type**: Bổ sung MIME type cho Blob để đảm bảo định dạng Content-Type trong các phần multipart được truyền tải chính xác.
- <kbd>98c1be5</kbd> **Tương thích FormData**: Chuyển đổi từ @RequestPart sang @RequestParam tại backend để tăng tính tương thích với định dạng FormData từ trình duyệt.
- <kbd>0aa4a0d</kbd> **Cải tiến phương thức upload**: Thay thế MultipartRequest mặc định bằng thư viện dart:html FormData và XHR để khắc phục lỗi tải tệp tin trên nền tảng Web.
- <kbd>caa8135</kbd> **Tối ưu hóa gọi API**: Bỏ qua client được tạo tự động cho tiến trình upload và sử dụng trực tiếp http.MultipartRequest để kiểm soát luồng dữ liệu tốt hơn.
- <kbd>221664e</kbd> **Xử lý Header**: Loại bỏ triệt để các header Content-Type bị trùng lặp trong yêu cầu HTTP.
- <kbd>4cbebaa</kbd> **Duy trì Boundary**: Đảm bảo giữ nguyên multipart boundary trong header Content-Type để tránh lỗi phân tích cú pháp dữ liệu.
- <kbd>18b4651</kbd> **Duy trì Boundary**: Đồng bộ hóa việc giữ nguyên multipart boundary cho các gói tin truyền tải.
- <kbd>250ef71</kbd> **Mở rộng giới hạn tải tệp**: Nâng cấp giới hạn dung lượng tệp tin multipart tại backend để hỗ trợ tải ảnh chất lượng cao.

### 🛠 Bảo trì hệ thống
- <kbd>f9367ee</kbd> **Định dạng mã nguồn**: Tự động định dạng toàn bộ mã nguồn bằng công cụ Spotless để đảm bảo tiêu chuẩn code chung.
