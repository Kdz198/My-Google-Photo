## 🎯 Tổng quan mục tiêu
Bản phát hành này tập trung vào việc hoàn thiện hệ thống quản lý media với tính năng tải ảnh lên, trích xuất metadata tự động và quản lý shortlink. Đồng thời, chúng tôi đã tối ưu hóa cấu trúc dữ liệu, cải thiện khả năng bảo trì và nâng cao trải nghiệm phát triển thông qua việc cấu hình môi trường linh hoạt và chuẩn hóa mã nguồn.

## ✨ Chi tiết thay đổi

### 🚀 Tính năng mới
- <kbd>a29680c</kbd> **Quản lý Shortlink**: Thêm mới các endpoint hỗ trợ danh sách (slice), chi tiết, xóa và xem trước shortlink.
- <kbd>a595400</kbd> **Trích xuất Metadata**: Tích hợp `MediaMetadataExtractor` để tự động lưu trữ thông tin EXIF vào entity Media khi tải tệp lên.
- <kbd>e16e648</kbd> **Tải lên Media**: Triển khai chức năng tải ảnh lên hệ thống bao gồm lưu trữ tệp tin và lưu trữ thông tin entity vào cơ sở dữ liệu.

### 🐛 Sửa lỗi
- <kbd>45a206c</kbd> **Cấu hình Swagger**: Điều chỉnh schema cho multipart upload sử dụng `@RequestPart` và annotation OpenAPI để hiển thị đúng tài liệu API.
- <kbd>17a45b3</kbd> **Cấu hình Java & Entity**: Cố định phiên bản Java 17 và tinh giản cấu trúc entity `AlbumMedia`.
- <kbd>f3b5869</kbd> **Hoàn thiện cấu hình**: Hoàn tất cấu hình môi trường `.env`, dọn dẹp logic `AlbumMedia` và cập nhật bổ sung thư viện kiểm thử H2.
- <kbd>21e56e7</kbd> **Cấu hình Môi trường**: Thiết lập file `.env`, thêm `DotenvConfig` và cấu hình datasource cho môi trường kiểm thử.

### 🛠 Bảo trì hệ thống
- <kbd>b7c35a2</kbd> **Refactor Exception Handling**: Chuyển logic xử lý lỗi vào `GlobalExceptionHandler` giúp code Controller trở nên tinh gọn.
- <kbd>7bd0489</kbd> **Thay đổi kiểu dữ liệu ID**: Chuyển đổi kiểu định danh (ID) từ UUID sang Long cho entity `Album` nhằm tối ưu hóa hiệu năng và quản lý.
- <kbd>75967a0</kbd> **Tự động hóa Timestamp**: Cập nhật các entity sử dụng `@CreationTimestamp` để tự động hóa việc ghi nhận thời gian tạo bản ghi.
- <kbd>846bdc0</kbd>, <kbd>a9aacd5</kbd>, <kbd>1adb716</kbd> **Định dạng mã nguồn**: Sử dụng Spotless để tự động hóa việc căn chỉnh và chuẩn hóa định dạng mã nguồn trên toàn dự án.
