## 🎯 Tổng quan mục tiêu
Bản phát hành này tập trung vào việc hoàn thiện cấu trúc quản lý đa phương tiện thông qua việc triển khai tính năng tải ảnh lên hệ thống và chuẩn hóa kiến trúc dữ liệu. Đồng thời, các cải tiến về hạ tầng như tự động hóa timestamp và tối ưu hóa cấu hình môi trường giúp nâng cao hiệu suất cũng như tính ổn định của hệ thống trong môi trường kiểm thử.

## ✨ Chi tiết thay đổi

### 🚀 Tính năng mới
- <kbd>e16e648</kbd> **Tích hợp tải ảnh**: Triển khai tính năng tải ảnh lên hệ thống, bao gồm lưu trữ file và thiết lập cơ chế persistence cho Media entity.
- <kbd>fc2aac4</kbd> **Khởi tạo dữ liệu Album**: Bổ sung các cấu trúc entity Album, AlbumMedia và repository tương ứng để quản lý bộ sưu tập ảnh.
- <kbd>95c2947</kbd> **Định danh UUID**: Bổ sung UUID cho AlbumMedia nhằm đảm bảo tính định danh duy nhất cho từng mục dữ liệu.

### 🛠 Bảo trì & Tối ưu hệ thống
- <kbd>7bd0489</kbd> **Thay đổi kiểu dữ liệu ID**: Chuyển đổi kiểu dữ liệu định danh từ UUID sang Long cho Album entity để tối ưu hóa truy vấn.
- <kbd>75967a0</kbd> **Tự động hóa Timestamp**: Cập nhật các entity classes sử dụng `@CreationTimestamp` để tự động ghi nhận thời gian tạo bản ghi.
- <kbd>f3b5869</kbd> **Hoàn thiện cấu hình kiểm thử**: Hoàn tất cấu hình môi trường (.env), làm sạch AlbumMedia và bổ sung H2 database phục vụ cho việc kiểm thử.
- <kbd>21e56e7</kbd> **Cấu hình môi trường**: Thiết lập DotenvConfig, cập nhật nguồn dữ liệu kiểm thử và thêm thư viện H2 dependency.
- <kbd>17a45b3</kbd> **Nâng cấp môi trường Java**: Cố định phiên bản Java 17 và tinh giản cấu trúc của AlbumMedia entity.
- <kbd>e3ff207</kbd> **Chuẩn hóa Media entity**: Hoàn thiện cấu trúc Media entity sau khi áp dụng các thay đổi từ Lombok và JdbcTypeCode.
- <kbd>b5dbab9</kbd> **Đồng bộ mã nguồn**: Hợp nhất các thay đổi mới nhất từ nhánh remote master.
- <kbd>eefa3fc</kbd> **Đồng bộ mã nguồn**: Hợp nhất các cập nhật bổ sung từ nhánh remote master.
