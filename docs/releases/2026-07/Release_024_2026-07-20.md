## 🎯 Tổng quan mục tiêu
Bản phát hành này tập trung vào việc hiện đại hóa cấu trúc dự án sang mô hình Monorepo để tối ưu hóa quản lý mã nguồn, đồng thời tích hợp frontend Flutter và bổ sung các tính năng xử lý hình ảnh quan trọng. Ngoài ra, chúng tôi đã hoàn thiện hạ tầng CI/CD và cấu hình bảo mật giúp nâng cao độ ổn định và trải nghiệm triển khai cho hệ thống.

## ✨ Chi tiết thay đổi

### 🚀 Tính năng mới
- <kbd>[80baa46]</kbd> **Tích hợp Frontend Flutter**: Khởi tạo cấu trúc dự án Frontend sử dụng Flutter trong mô hình Monorepo.
- <kbd>[7d8b207]</kbd> **Dịch vụ Thumbnail**: Bổ sung `ThumbnailService` để tự động tạo ảnh thu nhỏ định dạng JPEG kích thước 400x400 khi người dùng tải tệp tin lên.
- <kbd>[0eca387]</kbd> **Cấu hình CORS toàn cục**: Thiết lập cấu hình CORS trên backend để hỗ trợ giao tiếp an toàn giữa các dịch vụ.

### 🐛 Sửa lỗi
- <kbd>[1f219d3]</kbd> **Cấu hình bảo mật Swagger**: Điều chỉnh URL server HTTPS cho Swagger và cấu hình tin cậy reverse proxy headers.
- <kbd>[daca16a]</kbd> **Khôi phục bộ chọn tệp Swagger**: Thay đổi cơ chế từ `@RequestBody` sang `@Parameter` với format=binary để đảm bảo tính năng upload tệp trên giao diện tài liệu API hoạt động bình thường.

### 🛠 Bảo trì hệ thống
- <kbd>[6aa2c03]</kbd> **Chuyển đổi cấu trúc Monorepo**: Tái cấu trúc dự án thành mô hình Monorepo với phân tách rõ ràng giữa thư mục `backend` và `frontend`.
- <kbd>[15e10e3]</kbd> **Thiết lập CI/CD**: Cấu hình pipeline CI/CD tự động và bổ sung Dockerfile để chuẩn hóa quy trình triển khai.
- <kbd>[eb038d4]</kbd> **Tối ưu hóa dữ liệu media**: Cập nhật logic truy xuất bản xem trước (preview) sử dụng kiểu dữ liệu `MediaPreview` để tăng tính đồng nhất.
- <kbd>[1f166d6]</kbd> **Tích hợp thư viện UI/UX**: Thêm gói `ui-ux-pro-max-skill` để hỗ trợ phát triển các tính năng thiết kế thông minh từ AI.
- <kbd>[2bdd2e8]</kbd> **Tự động hóa định dạng mã nguồn**: Thiết lập Spotless để tự động định dạng mã nguồn, đảm bảo chuẩn code thống nhất trong toàn dự án.
