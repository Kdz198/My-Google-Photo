## 🎯 Tổng quan mục tiêu
Phiên bản này đánh dấu cột mốc khởi tạo dự án "My-Google-Photo" với nền tảng Spring Boot hiện đại, thiết lập môi trường phát triển chuẩn hóa thông qua tự động hóa. Việc tích hợp các công cụ hỗ trợ tài liệu API và định dạng code giúp nâng cao chất lượng mã nguồn và tối ưu hóa quy trình vận hành cho đội ngũ phát triển.

## ✨ Chi tiết thay đổi

### 🚀 Tính năng mới
- <kbd>[a06395d]</kbd> **Tích hợp Springdoc-openapi**: Thêm thư viện hỗ trợ tự động tạo tài liệu API chuẩn OpenAPI, giúp việc kiểm thử và tích hợp hệ thống trở nên thuận tiện hơn.
- <kbd>[3687d2d]</kbd> **Thiết lập công cụ định dạng mã nguồn**: Tích hợp Spotless để tự động hóa việc định dạng code, đảm bảo tính nhất quán trên toàn dự án.
- <kbd>[e3ffdd9]</kbd> **Khởi tạo dự án**: Thiết lập cấu trúc dự án cơ bản với Maven và Spring Boot làm nền tảng phát triển bền vững cho My-Google-Photo.

### 🛠 Bảo trì hệ thống
- <kbd>[577ee75]</kbd> **Cập nhật quy trình CI**: Tối ưu hóa các workflow của hệ thống CI để hỗ trợ tự động hóa việc định dạng mã nguồn và tạo ghi chú phát hành.
- <kbd>[d2f2e9c]</kbd> **Định dạng lại mã nguồn**: Thực thi tự động format toàn bộ code dự án theo chuẩn đã thiết lập để duy trì chất lượng mã sạch.
- <kbd>[6fcfc7c]</kbd> **Tự động hóa ghi chú phát hành**: Cấu hình quy trình tự động sinh release note giúp việc theo dõi các thay đổi trong phiên bản trở nên minh bạch và chuyên nghiệp.

### 🔄 Khác
- <kbd>[f918111]</kbd> **Đồng bộ hóa mã nguồn**: Cập nhật các thay đổi mới nhất từ nhánh master về nhánh phát triển hiện tại.
