## 🎯 Tổng quan mục tiêu
Bản phát hành này tập trung vào việc chuẩn hóa cấu trúc dự án theo mô hình Monorepo, tối ưu hóa quy trình vận hành thông qua CI/CD và mở rộng khả năng tích hợp hệ thống. Đồng thời, phiên bản này bổ sung giao diện quản lý ảnh và tăng cường tính bảo mật, linh hoạt cho cấu trúc Backend, mang lại trải nghiệm phát triển và vận hành hiệu quả hơn.

## ✨ Chi tiết thay đổi

### 🚀 Tính năng mới
- <kbd>1ce3e90</kbd> **Tích hợp Gallery**: Triển khai màn hình hiển thị thư viện ảnh chính với khả năng tích hợp API hoàn thiện.
- <kbd>80baa46</kbd> **Phát triển Frontend Flutter**: Bổ sung ứng dụng Flutter vào cấu trúc Monorepo để mở rộng nền tảng hỗ trợ.
- <kbd>6ad2430</kbd> **Tự động hóa OpenAPI**: Thiết lập CI và cơ chế tự động tạo client từ OpenAPI để đồng bộ hóa giao tiếp giữa Frontend và Backend.
- <kbd>1f166d6</kbd> **Tích hợp AI Design**: Bổ sung gói `ui-ux-pro-max-skill` nhằm hỗ trợ khả năng phân tích và thiết kế thông minh cho giao diện.

### 🛠 Bảo trì hệ thống
- <kbd>6aa2c03</kbd> **Cấu trúc lại Monorepo**: Tái cấu trúc toàn bộ dự án theo mô hình Monorepo với phân tách thư mục Backend và Frontend rõ ràng.
- <kbd>15e10e3</kbd> **Thiết lập CI/CD**: Bổ sung quy trình tự động hóa (CI/CD) và file Docker để tối ưu hóa quá trình build và deploy.
- <kbd>f9367ee</kbd> / <kbd>2bdd2e8</kbd> **Định dạng code**: Tự động chuẩn hóa định dạng mã nguồn bằng công cụ Spotless để đảm bảo tính nhất quán.

### 🐛 Sửa lỗi & Cấu hình
- <kbd>250ef71</kbd> **Điều chỉnh giới hạn tải file**: Nâng cao giới hạn kích thước cho các tệp tin dạng multipart để hỗ trợ upload dữ liệu lớn.
- <kbd>0eca387</kbd> **Cấu hình CORS toàn cục**: Thiết lập quy tắc CORS trên toàn hệ thống để đảm bảo việc kết nối giữa các domain an toàn và đúng quy chuẩn.
- <kbd>1f219d3</kbd> **Cấu hình HTTPS & Proxy**: Cấu hình URL HTTPS cho Swagger và thiết lập cơ chế tin tưởng reverse proxy headers để bảo mật kết nối.
