## 🎯 Tổng quan mục tiêu
Bản phát hành này tập trung vào việc thiết lập nền tảng dự án "My-Google-Photo" với cấu trúc Maven và Spring Boot, đồng thời chuẩn hóa quy trình phát triển thông qua việc tự động hóa định dạng mã nguồn và tài liệu hóa API. Những cải tiến này giúp tối ưu hóa trải nghiệm lập trình, đảm bảo tính nhất quán của mã nguồn và sẵn sàng cho việc mở rộng tính năng trong tương lai.

## ✨ Chi tiết thay đổi

### 🚀 Tính năng mới
- <kbd>0cb5732</kbd> **Cấu hình Database**: Thiết lập các thuộc tính kết nối cơ sở dữ liệu trong file `application.properties` để chuẩn bị cho việc lưu trữ dữ liệu.
- <kbd>a06395d</kbd> **Tích hợp Springdoc-OpenAPI**: Bổ sung dependency cần thiết để tự động hóa việc tạo tài liệu API chuẩn OpenAPI, hỗ trợ kiểm thử và tích hợp hệ thống dễ dàng hơn.
- <kbd>3687d2d</kbd> **Tự động hóa quy trình**: Tích hợp công cụ Spotless để định dạng mã nguồn tự động và thiết lập cơ chế tự động tạo ghi chú phát hành (release note).
- <kbd>e3ffdd9</kbd> **Khởi tạo dự án**: Thiết lập cấu trúc dự án cơ bản cho "My-Google-Photo" sử dụng Maven và framework Spring Boot.

### 🛠 Bảo trì hệ thống
- <kbd>577ee75</kbd> **Cập nhật CI Workflows**: Tinh chỉnh các luồng công việc CI để tự động hóa quy trình định dạng mã nguồn và tạo ghi chú phát hành ổn định hơn.
- <kbd>d2f2e9c</kbd> **Định dạng mã nguồn**: Thực hiện định dạng tự động toàn bộ mã nguồn theo chuẩn bằng công cụ Spotless để đảm bảo tính đồng nhất.
- <kbd>1da4f98</kbd> **Tự động cập nhật tài liệu**: Hệ thống tự động tạo nội dung ghi chú phát hành mới nhất.
- <kbd>6fcfc7c</kbd> **Tự động cập nhật tài liệu**: Hệ thống tự động cập nhật ghi chú phát hành cho các thay đổi mới nhất.
- <kbd>f918111</kbd> **Đồng bộ mã nguồn**: Thực hiện hợp nhất (merge) các thay đổi từ nhánh `master` để đảm bảo code base luôn ở trạng thái mới nhất.
